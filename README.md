FeelSense is a desktop GUI application that analyzes user emotions from text or voice input using sentiment analysis.

## Features
- Text-based sentiment analysis
- Voice input using microphone
- Real-time polarity score
- Mood-based motivational feedback
- Clean modern UI (CustomTkinter)

## Tech Stack
- Python
- CustomTkinter
- TextBlob
- SpeechRecognition
